# Role Description

The **Traveller Intelligence** track operates within Booking.com’s Marketplace organization and focuses on solving traveller needs across all platforms.

The team builds foundational capabilities that enable every product, experience, and AI-driven interaction at Booking.com to become:
- More personalized
- More consistent
- More customer-centric

At the core of the platform is a governed, real-time Traveller Profile and API gateway that unifies traveller context and behavioral signals into a single trusted ecosystem.

This platform powers:
- Product experiences
- AI agents
- Analytics systems
- Personalization engines

The goal is to create seamless traveller experiences while enabling smarter business growth through unified and actionable traveller intelligence.
