# Problem Statement: AI-Native Product Manager Career Transformation System

## Overview

The objective of this project is to build a highly personalized, AI-powered career transformation system that helps reposition an experienced Business Analyst profile into a strong, AI-native and AI-enabled Product Manager profile targeted toward high-growth startups and pre-IPO companies.

The system must go far beyond simple resume rewriting. It should function as an end-to-end Product Management career acceleration engine capable of:

- Reframing prior experience into product-focused narratives
- Identifying transferable PM competencies
- Positioning the candidate competitively for high-paying startup PM roles
- Optimizing all professional branding assets for ATS systems, recruiters, hiring managers, and founders
- Enabling highly targeted outbound networking and job search workflows

The final output should make the candidate appear as a high-impact, metrics-driven, AI-aware Product Manager with strong execution, analytical, operational, stakeholder management, and product thinking capabilities.
